Graphics Competitions  

Textmode Graphics Compo

You can use any tool you want to create a static image in text mode. ASCII, ANSI, Teletext, etc. All pictures will be shown with the latest version of Pablo Draw or IrfanView. Please provide screenshot of your entry to make sure we're displaying it properly or to use as the display if Pablo Draw or Irfanview can't handle it. If you can submit steps of the creation process, they will be shown.

Executable Graphics Compo

Same rules as Revision 4k graphics compo.

Photo Graphics Compo

Submit a photo you have taken. Only minor software retouch work allowed. All pictures will be shown with the latest version of IrfanView. Please hand in pictures with a flawless compression like PNG to ensure highest quality.

Oldschool Pixel Graphics Compo

You can use any tool or platform to create a pixel graphics entry. Please provide both the original format and a PNG snapshot/conversion of your image. All pictures will be shown with the latest version of IrfanView. If you can submit steps of the creation process, they will be shown.

Merge down rule

Compos with less then 3 entries submitted will be merged down into the closest matching category or an overall combined graphics compo.
     
Music Competitions  

.4ch .MOD Tracked Music Compo

You can use any tool you want to create a 4 channel .mod file which we'll play using xmplay. Max length 5 minutes.

.IT vs .XM Tracked Music Compo

You can use any tool you want to create an .it or .xm module which we'll play using xmplay. Max length 5 minutes.

Streaming Music Compo

You can use any tool you want to create a streaming music file. We accept mp3/ogg. Max length 5 minutes.

Oldschool Platforms Music Compo

Same rules as Revision 2020 Oldschool music compo. Max length 5 minutes.

Streaming Music Compo

You can use any tool you want to create a streaming music file. We accept mp3/ogg. Max length 5 minutes.

Executable Music Compo

Same rules as Revision 2020 executable music compo. Max length 5 minutes.

Mixed-in drone music

You've heard mixed in acid compos and mixed in dance music compos. This is the same but for drone music. Submit your drone / field recordings / dark ambient / shoegaze noise tracks. Any tracks with rythmic percussion elements or prominent high range sounds will be disqualified from the compo. Max length 15 minutes. All submissions will be mixed into one single continuous drone stream.

Merge down rule

Compos with less then 3 entries submitted will be merged down into the closest matching category or an overall combined graphics compo.
     
Wild Competitions  

Short Film Compo

You can use any tool you want to create a short film. We accept 3d animations, step motion, machinima, home videos, etc. We accept whatever format VLC can play out of the box, try to keep it small though. Max length 7 minutes.

Interactive Compo

Interactive thing for whatever platform. Videogame, instalattion project, whatever. Present it live on stage or with a pre-recorded video.

Demo Competition  

Demo Compo

Executable realtime audiovisual demo of maximum 7 minutes length. Make it play out of a vanilla Windows 10 machine with latest graphics drivers. For other platforms please let us know what free emulator. Please also submit a video capture of the intro to facilitate screening process, must play on VLC.

Submission System

Legalese

All works must be original and not prior released in public. The organizing team can refuse or disqualify your entry from participating in the competition if we deem it to be subjectively innapropriate (avoid insulting people and you should be fine, we just need a safeguard rule in case of extremely controversial content being delivered). By participating in our competitions you are giving us the right to redistribute your submitted work under Creative Commons, Non Commercial, No Derivatives, Share Alike license online.
     
Rules   In legal terms Inércia Demoparty is a closed doors private event for friends, ofcourse we welcome everyone interested in the demoscene and retro computing culture to show up, please pre-register so we know how many people we can expect to have. We reserve the right to refuse entry or kick out anyone with improper behavior such as racism, sexism, annoying drunks, infringment of Portuguese laws, etc. Remember we are a kid friendly event, please be sensible and minimize any explicit mature content whenever possible. Please remember to respect COVID-19 health recommendations of use of mask, hand sanitizer and social distancing. We hope these simple rules won't stop you from having fun at Inércia Demoparty 2020.
