Graphics Competitions  

Textmode Graphics Compo

You can use any tool you want to create a static image in text mode. ASCII, ANSI, Teletext, etc. All pictures will be shown with the latest version of Pablo Draw or IrfanView. Please provide a PNG screenshot of your entry to make sure we're displaying it properly or to use as the display if Pablo Draw can't handle it properly. If you can submit steps of the creation process, they will be shown (also PNG format please).

Executable Graphics Compo

Same rules as Revision 4k graphics compo. Maximum resolution 1920x1080.

Photo Graphics Compo

Submit a photo you have taken. Only minor software retouch work allowed. All pictures will be shown with the latest version of IrfanView. Please hand in pictures with a flawless compression like PNG to ensure highest quality. Maximum resolution 1920x1080.

Oldschool Pixel Graphics Compo

You can use any tool or platform to create a pixel graphics entry. Please provide both the original format and a PNG snapshot/conversion of your image. All pictures will be shown with the latest version of IrfanView. If you can submit steps of the creation process, they will be shown.

Modern Graphics Compo

Any method allowed as long as you have a PNG snapshot that we can show. All pictures will be shown with the latest version of IrfanView. If you can submit steps of the creation process, they will be shown. Maximum resolution 1920x1080.

Merge down rule

Compos with less then 3 entries submitted will be merged down into the closest matching category or an overall combined graphics compo.
     
Music Competitions  

.4ch .MOD Tracked Music Compo

You can use any tool you want to create a 4 channel .mod file which we'll play using xmplay. Max length 5 minutes. Please also submit the mp3 of your entry to facilitate correct playback.

.IT vs .XM Tracked Music Compo

You can use any tool you want to create an .it or .xm module which we'll play using xmplay. Max length 5 minutes. Please also submit the mp3 of your entry to facilitate correct playback.

Oldschool Platforms Music Compo

Same rules as Revision 2020 Oldschool music compo. Max length 5 minutes. Please also submit the mp3 of your entry to facilitate correct playback.

Streaming Music Compo

You can use any tool you want to create a streaming music file. We accept mp3/ogg. Max length 5 minutes.

Executable Music Compo

Same rules as Revision 2020 executable music compo. Max length 5 minutes. Please also submit the mp3 of your entry to facilitate correct playback.

Mixed-in Drone Music

You've heard mixed in acid compos and mixed in dance music compos. This is the same but for drone music. Submit your drone / field recordings / dark ambient / shoegaze noise tracks. We accept mp3/ogg. Any tracks that deviate too much from definition of drone music genre will be disqualified! Max length 15 minutes. All submissions will be mixed into one single continuous drone stream.

Merge down rule

Compos with less then 3 entries submitted will be merged down into the closest matching category or an overall combined music compo.
     
Wild Competitions  

Mixed-in Drone Footage Video

This competition accepts any flying drone video footage. Please respect your local privacy and drone flight safety laws. Only submit videos to which you have copyright ownership of. The videos will be used as visuals during the mixed-in drone music competition.

Short Film Compo

You can use any tool you want to create a short film. We accept 3d animations, stop motion, machinima, home videos, etc. We accept whatever format VLC can play out of the box, try to keep it small though. Max length 7 minutes.

Interactive Compo

Interactive thing for whatever platform. Videogame, installation project, whatever. Present it live on stage or with a pre-recorded video.

Demo Competition  

Demo Compo

Executable realtime audiovisual demo of maximum 7 minutes length. Make it play out of a vanilla Windows 10 machine with latest graphics drivers. For other platforms please let us know what emulator and settings to use. Please also submit a video capture of the intro to facilitate screening process, must play on VLC.
